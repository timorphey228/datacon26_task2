## Минитаск 2 для датакона 
В extractor лежит класс, выполняющий все функции по экстракции \
Для любой статьи можно добавить конфиг в папку с ней, что позволяет гибко настраивать экстракцию и использовать в дальнейшем любые статьи \
В качестве примера я взял только 1 статью, так как все вместе это бы слишком много весило, но я думаю результат понятен и отражает возможности кода \
Код сохраняет различные значения, полученные из текста, таблицы с данными, читает графики и переводит их в те же таблицы и сохраняет изображения. \
